# AVR Programming

Uploading HEX code to arduino using avrdude:

avrdude -v -p atmega328p -c arduino -P /dev/ttyUSB0 -b 57600 \
  -D -U flash:w:blink.hex:i

Uploading HEX code to ATTiny microcontroller:

avrdude -c usbtiny -p t2313 -U flash:w:file.hex:i

# Set Fuse for virgin ATMega8A-AU

avrdude -p m8 -c usbasp -P  -F -u -b 57600  -B 400  \
-U lfuse:w:0xd4:m -U hfuse:w:0xd1:m

The key parameter is -B 400.

Make sure voltage jumper in USB-ASP in 5 volt mode.
