# Cryptography

# Command Examples

Generate protected RSA private key:

openssl genrsa -aes128 -out app-server-001-priv.pem 4096

Get RSA public key:

openssl rsa -outform PEM -out app-server-001-pub.pem -pubout \
  -in app-server-001-priv.pem

Decrypt a ciphertext file

echo string64 | base64 -d | openssl rsautl -decrypt -oaep -out zzz.txt \
  -inkey /data/digital-key/apps/app-server1.pem

Convert encrypted RSA private key to unencrypted PKCS8 private key (needed for Web Cypto API):

openssl pkcs8 -topk8 -inform PEM -outform PEM -nocrypt \
  -in pkcs1-protected.pem \
  -out pkcs8-unprotected.key

Convert encrypted RSA private key to unencrypted RSA private key:

openssl rsa -in protected.pem -out unprotected.pem

Show Contents Of An X509 Certificate

openssl x509 -in cert1.pem -text -noout

Use jwk2pem.py Python script to convert JWK to RSA PEM format.

import json
import jwcrypto.jwk
import sys

f = open(sys.argv[1], "r")
d = json.load(f)
f.close()
key = jwcrypto.jwk.JWK(**d)
f = open(sys.argv[1].replace(".json",".pem"), "w")
f.write(key.export_to_pem(private_key=False, password=False).decode("utf-8"))
f.close()

To convert DER file to PEM format:

openssl rsa -inform DER -outform PEM -pubin -in public.der -out public.pem

To convert PEM file to DER format:

openssl rsa -inform PEM -outform DER -pubin -in public.pem -out public.der

# Private Certificate Authority (CA)

# Key Algorithm

  • RSA 2048: This is widely supported by browsers and client. This provides good ballance between security an efficiency. Default.
  • RSA 4096: This is less efficient than RSA 2048 and used for root CA.
# Extract public key from x509 certificate
openssl x509 -in cert.pem -keyform PEM -pubkey -noout > /tmp/pubkey.pem

# Sign a file
openssl dgst -sha256 -sign key.pem -out /tmp/signature.bin data.txt

# Verify a file based on public key and a signature file
openssl dgst -sha256 -verify /tmp/pubkey.pem -signature \
  /tmp/signature.bin data.txt
