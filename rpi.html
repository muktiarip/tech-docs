# Raspberry Pi

# How to setup WiFi on your Raspberry Pi

https://howchoo.com/g/ndy1zte2yjn/how-to-set-up-wifi-on-your-raspberry-pi-without-ethernet

Create a file in this directory called /boot/wpa_supplicant.conf. The file should contain the following details:

For Raspbian Stretch and Raspbian Buster:

country=ID
update_config=1
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
network={
    ssid="YOUR_NETWORK_NAME"
    psk="YOUR_PASSWORD"
    key_mgmt=WPA-PSK
}

# Cross Compilling With Rust Programming Language

Raspberry Pi 3 Target:

rustup target add armv7-unknown-linux-gnueabihf
sudo apt install gcc-8-multilib-arm-linux-gnueabihf
cargo build --target=armv7-unknown-linux-gnueabihf
cargo build --release --target=armv7-unknown-linux-gnueabihf

# Solution for excessive logging of RTL 8188EU driver in Raspbian Buster:

Edit file /etc/modprobe.8188eu.conf and add rtw_drv_log_level=0 to the option list.
